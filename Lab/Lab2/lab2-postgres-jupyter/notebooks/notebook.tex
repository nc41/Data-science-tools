\section{Lab 2 - Postgres}\label{lab-2---postgres}}

In this lab we will demonstrate using Postgres through a Jupyter
Notebook which you will all be doing for Assignment 1 and 2.

    \hypertarget{cyphers}{%
\subsection{Cyphers}\label{cyphers}}

For this lab, you are given a simple substitution cypher (e.g.'a' = 5,
`b' = 1, etc.) and a secret message that has been encoded using the
cypher. You will need to create tables to store the cypher and the
secret message. Next you will load the provided cypher and secret
message into the tables and write a query to decrypt the secret message.

For example, if our cypher is:

\begin{verbatim}
a = 2
b = 3
c = 1
\end{verbatim}

and our secret message is `1 2 3', then the decrypted message is: `cab'.

    \hypertarget{table-setup}{%
\subsection{Table Setup}\label{table-setup}}

At this point, you should have created these two tables in your
database. If not, go back and follow the instructions in the lab
document.

\begin{itemize}
\tightlist
\item
  cypher: This table will store the cypher by which our message has been
  encrypted. It has two attributes

  \begin{itemize}
  \tightlist
  \item
    letter, which has datatype character, and contains the symbol to be
    encoded
  \item
    value, which has datatype integer. This attribute contains the
    integer used to represent letter in the cypher
  \end{itemize}
\item
  message This table will store the secret message. It also has two
  attributes

  \begin{itemize}
  \tightlist
  \item
    seq, which has datatype integer. This attribute indicates where in
    the message this symbol is found
  \item
    letter, which has datatype integer. This attribute contains the
    integer that the symbol in the original message is mapped to by the
    cypher
  \end{itemize}
\end{itemize}
